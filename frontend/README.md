## Tech Stack

> Backend

- Language: `Typescript`
- Framework: `React Vite`
- CSS Framework: `Tailwind`
- API Connection: `Axios`

## Installation

```ps
cd frontend
```

### 1. Create `.env` file

```
VITE_BASE_API_URL=http://localhost:5127
VITE_NOREK=123456789012
```

### 2. Install Dependencies

```ps
npm install
```

### 3. Build App

```ps
npm run build
```
