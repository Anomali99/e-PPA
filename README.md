## Tech Stack

> Backend

- Language: `Python 3.12.3`
- Framework: `Flask 3.0.3`
- ORM: `SQLAlchemy 2.0.31`
- Database: `MariaDB 11.3.2`
- Authentication: `Flask-HTTPAuth 4.8.0`

> Frontend

- Language: `Typescript`
- Framework: `React Vite`
- CSS Framework: `Tailwind`
- API Connection: `Axios`
